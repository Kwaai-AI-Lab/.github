## Updating the Landing Page

To update the organization profile:

1. Clone this repository
2. Edit `profile/README.md`
3. Follow the existing structure and formatting
4. Submit a pull request
5. Get review from team members
6. Merge to main branch

Changes to `profile/README.md` are reflected immediately at https://github.com/Kwaai-AI-Lab

## Structure

The landing page organizes all repositories by official work tracks:

1. **Personal AI Development Track**
   - Personal AI Agent Workgroup (Led by Balaji Lakshmanan)
   - KwaaiNet DePIN Workgroup (Led by Brian Ragazzi)
   - Personal Agency Workgroup (Led by Steve Vitka)
   - User Experience & Applications

2. **Fundamental AI Research Track**
   - State Space Models (Led by Daryle Serrant)
   - AI 4 Med (Led by Diego Galeano)
   - Physics Inspired (Led by Reza Rassool)
   - Homomorphic Encryption (Led by Sulimon Sattari)
   - Distributed Knowledge Base & Beyond Graph RAG
   - AI Alignment Research

3. **Policy & Governance Track**
   - AI Policy and Alignment (Led by Steve Vitka)
   - AI Trust "Loyal Agents"
   - "MyTerms" IEEE P7012

Plus **Supporting Infrastructure** for DevOps and operational tools.

## Badge System

Each repository includes:
- **Language badge**: Primary programming language
- **Status badge**: Development status (Production, Active Development, Beta, Prototype, Research, Legacy)
- **Visibility badge**: Public (blue) or Private (red)
